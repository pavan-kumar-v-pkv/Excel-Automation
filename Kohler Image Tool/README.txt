## What's This?

A simple tool to automatically insert Kohler product images into your Excel quotation files.

---

## What You Get

This folder contains:

- **Kohler Image Tool.bat** - For Windows (double-click to use)
- **Kohler Image Tool.command** - For Mac (double-click to use)
- **Instructions.md** - How to use it (start here!)
- **Excel Macro - Summary Generator.txt** - Optional macro for creating summary sheets

---

## Quick Start

### 1. First Time Setup (Fully Automatic!)
- **Windows:** Double-click **Kohler Image Tool.bat**
  - Python installs automatically (2-3 minutes)
  - Packages install automatically
  - GUI opens automatically - Ready to use!
  
- **Mac:** Double-click **Kohler Image Tool.command**
  - Follow the one-time Python installation prompt
  - Packages install automatically
  - GUI opens - Ready to use!

### 2. Use the Tool (Every Time After)
- **Windows:** Double-click **Kohler Image Tool.bat**
- **Mac:** Double-click **Kohler Image Tool.command**
- Select your Excel file
- Select your PDF catalog
- Click START
- Done!

---

## Excel Macro (Optional)

Want to auto-generate summary sheets in Excel?

1. Open **Excel Macro - Summary Generator.txt**
2. Follow the instructions in **Instructions.md**
3. Run the macro in Excel whenever needed

---

## System Requirements

- **Windows 10/11** or **macOS 10.12+**
- Internet connection (first time only)
- Excel file format: .xlsx or .xlsm (NOT .xlsb)
